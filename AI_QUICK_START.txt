STEP 4: Open in Browser
────────────────────────────────────────────────────────────────────────────
Go to: http://localhost:5173

STEP 5: Use the AI Resume Generator
────────────────────────────────────────────────────────────────────────────
1. Click "🤖 AI Generate" tab on the left panel
2. Paste a job description in the textarea
3. Click "Generate with AI" button
4. Review the generated content in the preview
5. Click "Apply to Resume" to merge it into your resume
6. Switch back to "📝 Manual Entry" tab to review merged data
7. Click "Preview PDF" to see the final resume
8. Click "Download PDF" to save the file

  📊 SYSTEM ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

Frontend (React):
  AIResumeGenerator.jsx → User inputs job description
  ↓
  useResumeGenerator.js → Sends HTTP POST request
  ↓
  Backend API Server (Express) on port 3001
  ↓
  OpenAI API (GPT-3.5-turbo) → Generates resume content
  ↓
  JSON Response → { summary, skills[], experience[] }
  ↓
  Home.jsx → Merges with existing form data
  ↓
  Preview.jsx → Shows updated resume
  ↓
  PDF Export → Download or print

═══════════════════════════════════════════════════════════════════════════════
  🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "AI service is not configured"
Solution: Check .env file has OPENAI_API_KEY set

Problem: Backend won't start
Solution: Make sure port 3001 is not in use
         netstat -ano | findstr :3001 (Windows)
         lsof -i :3001 (Mac/Linux)

Problem: Slow response or timeout
Solution: Job descriptions >2000 characters take longer
         Check internet connection
         Verify OpenAI API status: https://status.openai.com/

Problem: Generated content is empty
Solution: Try again - API calls can be flaky
         Check browser console (F12) for detailed errors
         Rephrase the job description more clearly
